## PathFinder ­ЪЌ║№ИЈ

**PathFinder** рЃљрЃарЃўрЃА рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃў **React**РђЊрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃљрЃЊ.
рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃњрЃќрЃўрЃА рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА A\*, BFS, DFS рЃЊрЃљ Dijkstra рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃЌ, рЃљрЃАрЃћрЃЋрЃћ рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃЌ.

­ЪїЇ **рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ:** [PathFinder (Render)](https://path-finder-ouhx.onrender.com/)

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### Client side (React)

- рЃњрЃќрЃўрЃА рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (A\*, BFS, DFS, Dijkstra)
- рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ (Aldous-Broder, Binary Tree, Sidewinder)
- рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃњрЃарЃўрЃЊрЃў рЃБрЃ»рЃарЃћрЃЉрЃќрЃћ рЃќрЃћрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ
- рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
- рЃЏрЃћрЃюрЃўрЃБ рЃЊрЃљ рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃў рЃњрЃќрЃўрЃА рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА

## РџЎ№ИЈ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

### 1. рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ

```
bash
  git clone https://github.com/NikaMalakmadze/Path-Finder.git
  cd Path-Finder
```

### 2. рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

```

  npm install
```

### 3. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

  npm run dev

## ­ЪЊї рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљ рЃљрЃарЃљ **production-ready** рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ.

рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ **React**, **Redux**, рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃќрЃўрЃА рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃљрЃЊ
